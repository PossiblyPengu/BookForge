    How it works

    1. Drop your MP3 chapters in playback order.
    2. Fine-tune order or remove tracks if needed.
    3. Click “Compile to M4B” to merge & convert.
    4. Download the finished .m4b file once the process completes. Everything stays in your browser — nothing is uploaded to a server.

    Powered by FFmpeg.wasm — the first load downloads ~31 MB of WebAssembly. Subsequent visits use the browser cache.
